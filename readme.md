# рџ«Ў Setup & Run Locally

## 1. Clone the Repository

```bash
git clone https://github.com/princeraj07m/o2H_Mock_Assessment.git
cd o2H_Mock_Assessment
```

## 2. Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd frontend
npm install
```

## 3. Run the Backend

```bash
cd backend
npm run dev
```

Backend runs on:

```text
http://localhost:8000
```

## 4. Run the Frontend

```bash
cd frontend
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
